写在2017

发表于 2017-11-16
你要有里程碑；你要赋予自己的行动或作品或实践仪式感。有清晰的开始、结束，这样你才能一次又一次提高自己的行动力。

离去

2016年11月15日，我提出辞职了。与以往不同，这次辞职不像上次，是为了去另外一个公司从事相同的行业谋求更好的职位，而是彻底的告别原来的行业，一切从零开始的蜕变。提出辞职的时候，我的心里非常平静，没有一丝留恋，甚至是开心和喜悦的。当时真想原地起跳，在蔚蓝的空中向着星辰挥舞臂膀。那一刻，我意识到，谢天谢地，我重生了。

从大学到提出辞职打算转行，仔细一回想，是2010年到2016年，也就是说我在会计行业整六年。但这六年会计这个角色并未给我太多美好回忆，反而现在记忆犹新的是痛苦，那种现实感受和自己内心真正的渴求之间的挣扎：

  1. 进入大学时，专业是调剂的。并不是自己真正想学的专业。大学时读书还算认真，考出许多很硬的证件，但都和会计无关。上会计专业课时，听着并不是那么吸引人，这是专业课，你也得学，但吸引我的是更大的世界，有时间我就学别的。就这样快毕业了，开始找工作。找工作对我来讲还是蛮快的，虽然是考研失利后才开始的，但是因为简历上面的资质比较过硬，所以匆忙间就找到了一份工作。当时就想，学这个就干这个吧，再说那个时候没有任何工作经验，也不知道这个会计工作性质是什么样的，其他岗位像翻译、教师、市场等工作性质是什么样的。我只是大概知道，像其他人的对会计的传统印象那样：管钱管账，按部就班，坐办公室，与数字打交道。
  2. 但是其实，真正你做这个事情几年下来，你会发现，管钱管账，按部就班，坐办公室，与数字打交道的真正意思是：工作单一任务繁重，无穷无尽的简单重复劳动，一堆堆的单据，忙到没有自己的时间和生活。枯燥的工作，在我看来，我所做的只是简单的处理数字而已，没有任何成就感，没有任何创造性。这让我郁闷，这样我痛苦，我喜欢解决问题，而这个行业工作只是重复，月月重复，年年重复。所用的工具也非常死板，死气沉沉。
  3. 没有时间和环境去和人打交道，因为有那么多事情要做。越不接触社会，越不会社交，脾气也越怪，越信息闭塞，人也变得死板与较真，恶性循环就这样开始。每天就像那摩登时代里的卓别林。那时我就意识到，如果我投入更多的时间和公司之外的人际关系上，我反而会更快乐。从进入会计行业起，六年来从来没休过国庆假期，如果赶上了春节是月初的时候，那不好意思，大年三十你也要加班结账，月底，年底更是忙的焦头烂额，管你是周末还是法定节假日，加班结账。
  4. 人和其所处的环境产生相互的影响，形成的结果也是难以预料的。工作和社会关系成了我们生活中不可或缺的一部分，工作是一个人的个人价值和社会价值的重要来源。但是我并未在做会计这件事情上找到什么价值，反而会计工作性质决定的谨小慎微，较真刻板，斤斤计较，会让人变得很怪，对别人很不宽容。我想要的是去更多去考虑宏观长远的事情，去思考跟深层次的东西，而不是把有限的精力浪费在一分两分钱的差异、弄平报表以及字体排版上等琐事上。
  5. 行业前景惨淡，人工智能最先取代的就是会计这种规律性、制度性极强的工种。机器算的比人快，还比人准确。你看现在的自动柜员机的发展情况，银行柜员都不需要几个人了大量裁员，因为很多业务都可以在机器上完成了。
  6. 个人恩怨及不公正待遇，具体的经过就不写了，女上司大妈太狗血太极品。但是，这些事情中有类似的东西，是这个行业的共性问题。这些共性的问题说明，我跟会计这个行业的小喽啰“八字不合”。

不在继续做会计这个职业的火苗，在心里一点点越烧越旺。最终在我接触了内部动机理论之后彻底爆发，以至于一分钟都不想做跟会计有关的任何事情，很快我就辞职了。如果说人生的前三个十年，是我无法自己选择的，从事会计行业工作只是是机缘巧合，而非内心的选择。那么，在我即将迈入30岁的2016年，我自己选择人生：不再做会计。

归来

刚过完了三十岁生日没几天，2017年就接近尾声了，这也就意味着离开原来的行业也就近一年了。是时候写些东西，记录一下那段坎坎坷坷跌跌撞撞的十年时光。还原一下那日子的经历和感受，算是对自己过去的一个总结和告别。年初制定年度计划中有这个任务，之前因为忙于现在的事业就一直没有动笔，心里也一直惦记着要写完。

心里万句，下笔千言。语言能够表达的出的的自己，远远不及自己的期望。我们总有一腔热情想要说，却常常不知从何说起；我们一边渴望别人理解，一边又总觉得自己永远不能能表达出我们的全部。在我看来，其实大可不必。道可道，非常道；名可名，非常名，先贤早就讲明白了：语言和文字，作为进化后期才出现的一种表达自我的方式，对于我们百万年进化而来的强大的五官感受而言，是在太渺小了，有些感受是你永远无法表达出来的，但是你知道，他在那。当有的人问你，你为什么爱我，你可能说不出来答案，那又有什么关系，只要你知道你是爱的就可以了。至于我为什么爱，我还说不出，但是我的大脑，我的潜意识，我的第六感告诉我这是真的，这就够了。因为我能感觉的到。

所以这篇，我不求多么精准度表达自己，能写出一些感受和心得，能在下一个十年的某个时间可以回味，这就够了。
