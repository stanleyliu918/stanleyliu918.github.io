如何设置Mac Finder 文件夹图标默认显示在排列最上方

发表于 2017-11-27
Mac maxes modes.

从Windows laptop转投Macbook 近一年了。这几天把Finder中如何设置，使文件夹保持在最上面显示问题解决了。蛮开心。
方法很简单：

  1. 打开finder，进入任意一个文件目录位置。为了便于直观看到设置成功后的效果，建议进入带有文件夹和文件的位置。例如：
  2. 点击屏幕左上角Finder,进入偏好设置页面。
  3. 将“按名称排序时保持文件夹在顶部”前打上对号后关闭即可。
  4. 回过来看Finder界面，已经变成整齐的格式了
    而且，这个是针对所有Finder界面生效的哦。
